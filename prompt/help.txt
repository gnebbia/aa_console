================================================
[ COMMAND ]                [ RESULT ]
================================================
> list [type]                
     all                  - show all profiles
     enforce              - show enforce
     complain             - show complain
     help                 - show usage

> search [string]         - search for a
                            profile matching
                            string
     help                 - show usage

> generate [name] <flag>  - generate prof
                            with given name
     [name]  -m           - launch generate
                            in manual mode
     help                 - show usage

> log_search <flag>       - search for DENIED
                            processes in logs
     -t                   - search on the
                            day's log
     -w                   - search on the
                            weeks log

> help                    - print help
> exit                    - exit
